## Cách chạy dự án

Để chạy dự án này trên máy local:

```bash
# Clone repo về máy
git clone https://github.com/Cong-Chau/landing-page

# Di chuyển vào thư mục dự án
cd landing-page

# Cài đặt dependencies
npm install

# Chạy dự án
npm run dev
```
